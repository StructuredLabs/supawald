Open Source Headless CMS, built for Supabase

Manage Supabase Storage with a powerful, React-based CMS. Fully headless, beautifully designed, endlessly customizable.

Book a Demo Star on GitHub

Features

Intuitive File Management

Drag & drop file uploads, folder navigation, and in-place file editing with a modern, responsive interface

Modern Tech Stack

Built with Next.js 14, TypeScript, and Tailwind CSS for a robust developer experience

Secure Access Control

Basic auth protection and Supabase's Row Level Security for granular file access control

Real-time Updates

Live file changes with Supabase subscriptions and instant UI updates

Markdown Support

Built-in markdown preview and editing with frontmatter support for content management

Static Site Integration

Publish API for triggering static site regeneration when content changes

Flexible Storage

Support for both public and private Supabase Storage buckets with customizable policies

Version Control

File history tracking and last modified timestamps for better content management
